# Horizonte Tech

Este é o projeto da página inicial da **Horizonte Tech**, uma plataforma educacional focada no ensino de tecnologias da informação. O site apresenta informações sobre cursos de Desenvolvimento de Sistemas, Informática e Redes, além de um sistema de cadastro para interessados.

## 🚀 Funcionalidades

- **Carrossel de Imagens Dinâmico**: Uma vitrine visual que destaca as áreas de atuação (Sistemas, Informática e Redes).
- **Navegação Intuitiva**: Menu de acesso rápido para a página inicial, lista de cursos e formulário de inscrição.
- **Design Responsivo**: Estrutura preparada para visualização em diferentes dispositivos (mobile e desktop).
- **Seção Informativa**: Conteúdo detalhado sobre a proposta da instituição e os benefícios para a carreira do aluno.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização visual (incluindo reset de estilos e folha de estilos principal).
- **JavaScript**: Lógica de funcionamento do carrossel de imagens.

## 📂 Estrutura do Projeto

```text
atividade-aula-frontend-main/
├── assets/
│   ├── images/       # Imagens ilustrativas dos cursos
│   ├── scripts/      # Lógica JavaScript (carrrossel.js)
│   └── styles/       # Arquivos CSS (reset.css e main.css)
├── pages/
│   └── formulario.html # Página de cadastro de usuários
├── index.html        # Página principal do projeto
└── README.md         # Documentação do projeto
```

## 🔧 Como Executar

1. Faça o download ou clone este repositório.
2. Navegue até a pasta raiz do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

---

### Observação Técnica
O projeto utiliza caminhos relativos para ativos (imagens e scripts) e caminhos absolutos/relativos para CSS, garantindo que a estrutura funcione corretamente em ambientes de servidor local ou hospedagem web.
